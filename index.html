Projects


Automatic Identification of Global Features and Local Features of the Human Heart

NATIONAL HEART AND LUNG INSTITUTE | TAN, L; MIELEWCZIK, M; DHUTIA, N; FRANCIS, D

I created a fully-automated, vendor independent library to identify cardinal landmarks (mitral and tricuspid valve annuli) on moving echocardiographic images using a spatio-temporal-variance approach with analysis of greyscale histograms. I then validated both robustness and accuracy of several different imaging processing approaches in identifying these points.

Automation of the imaging analysis process for echocardiography has the potential to improve efficiency and accessibility. My program is able to identify initial landmarks in an operator-independent manner to then allow further workflows to run and enable automatic quantification of cardiac structure and function.

Exhibited as a poster presentation at the IC Biomedical Symposium 2017 (Hammersmith Hospital) and the Cardiovascular Undergraduate Conference 2018 (Imperial College London).




A robust and fast algorithm for fully automated detection of the ultrasound cone in echocardiographic image sequences

NATIONAL HEART AND LUNG INSTITUTE | MIELEWCZIK, M; TAN, L; DHUTIA, N; FRANCIS, D

To achieve accurate analysis of ultrasound images, an important first step is identifying the cone shaped image sector to eliminate unnecessary processing of irrelevant regions.

I assisted in creating a robust and reliable program using MATLAB, able to detect the active ultrasound image sector fully automatically within seconds. I then tested four different implementations of this program and evaluated them for robustness and reliability, achieving 100% accuracy in a large set of multi-beat imaging sequences comprising image qualities from low to high.
